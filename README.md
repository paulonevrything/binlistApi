# BinlistApi

This is an API wrapper for [binlist.net](https://binlist.net/). Written in .NET Core

## Installation

1. Restore Nugget

```bash
dotnet restore
```

2. Build

```bash
dotnet build
```

3. Run

```bash
dotnet run
```
